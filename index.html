Public-Key Encryption Demo

Learn how asymmetric encryption works with this interactive demonstration

1

Generate Key Pair

Generate a pair of mathematically related keys. The public key can be shared openly, while the private key must remain secret.

2

Encrypt Message

Enter a message and use one of the generated keys to encrypt it. Anyone can use your public key to encrypt messages for you.

3

Decrypt Message

Use the corresponding key to decrypt the message. If you encrypted with the public key, use the private key to decrypt (and vice versa).

Hint: If you used one of the keys from Step 1 to encrypt, the other key will decrypt the message.
